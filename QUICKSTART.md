# Relay - Quick Start Guide

## 🎯 What is Relay?

Relay is a browser-based collaboration tool that lets you share your screen, files, and notes with others on the same local network - **no internet required, no server setup needed!**

## 🚀 Getting Started (2 minutes)

### Step 1: Extract & Open

1. Extract the `relay-complete.zip` file
2. Open `index.html` in a modern browser (Chrome, Edge, Firefox)

### Step 2: Create or Join

**To Host:**
- Click "Create Room (Host)"
- Share the displayed room code with others
- Example code: `192-ABC`

**To Join:**
- Enter your name
- Enter the room code from the host
- Click "Join as Participant"

### Step 3: Collaborate!

**Host can:**
- Click "Start Sharing" to share screen
- Drop files into the file zone
- Create quick polls
- Write shared notes (click "Notes" button)

**Participants can:**
- View the live screen
- Save slides manually or enable auto-capture
- Download all saved slides
- Vote on polls
- Read shared notes

## 💡 Tips

### For Best Results
- Use Chrome or Edge for best compatibility
- Keep all devices on the same WiFi network
- Share presentation/slide mode, not entire screen
- Enable auto-capture to save all slides automatically

### Common Tasks

**Save a Slide (Participant)**
- Click "Save This Slide" button
- Or enable "Auto-capture" toggle

**Share a File (Host)**
- Drag and drop into the file zone
- Or click the zone to browse

**Create a Poll (Host)**
- Type your question
- Enter 2-6 options
- Click "Start"

**View Notes**
- Click "Notes" button (top right)
- Host can edit, participants can view

## 🔧 Troubleshooting

**Can't Join Room?**
- Make sure you're on the same WiFi as the host
- Check the room code is correct (case-sensitive)
- Try refreshing the page

**Screen Not Showing?**
- Host must click "Start Sharing"
- Grant screen share permission when prompted
- Select the correct window/screen

**Slides Not Saving?**
- Enable auto-capture toggle
- Or click "Save This Slide" manually
- Check browser allows local storage
